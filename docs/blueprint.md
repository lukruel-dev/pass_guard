# **App Name**: PassGuard

## Core Features:

- User Authentication: Secure login via Google or traditional email/phone and password, protecting access to stored passwords.
- Password Entry Management: Create, store, and manage entries for different websites or applications, such as Instagram, Facebook, etc.
- Dynamic Password Generation Tool: Generate strong, random passwords using user-defined criteria for length, number of special characters, uppercase, lowercase letters, and numbers.
- Password Interaction: Allow users to view, regenerate, and securely copy generated or stored passwords.
- Secure Password Storage: Encrypt and securely store all password entries in a persistent database.

## Style Guidelines:

- Dark scheme for a sense of security and professionalism. Primary color: a deep, vibrant blue (#477DEB) that evokes trust and clarity. This color will be used for interactive elements and key headings.
- Background color: a very dark, subtle blue (#21242C), offering high contrast and minimizing eye strain for prolonged use, reinforcing a modern and secure interface.
- Accent color: a rich purple (#8F7DE8), analogous to the primary blue, used for subtle highlights and call-to-action elements, providing a sophisticated contrast.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean, and highly legible appearance, ensuring clarity for all password details and application text.
- Use clear, concise security-themed icons (e.g., lock, shield, eye) that are intuitive and consistent with the application's purpose of managing sensitive information.
- A clean, minimalist layout focused on functionality. Employ card-based elements for password entries, ensuring easy scanning and management. Input forms will be straightforward and uncluttered.
- Subtle and quick animations for feedback, such as a brief visual confirmation when a password is copied or successfully generated, enhancing user experience without distraction.
